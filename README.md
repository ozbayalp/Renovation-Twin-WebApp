# Renovation-Twin-WebApp
WebApp that uses COLMAP to transform  building photos into 3D models with a heat map for damages and cost estimation.

## Branch: uiux-redesign

This branch is dedicated to a comprehensive UI/UX redesign of the web application. All new user interface, design system, and user experience improvements should be made here before being merged into main.

### Goals
- Modernize the look and feel of the app
- Improve usability, accessibility, and responsiveness
- Implement a consistent design system and visual language
- Prototype and test new flows before merging to main

### Workflow
- All UI/UX changes and experiments are committed to `uiux-redesign`
- Keep `main` stable and production-ready
- Merge from `main` to `uiux-redesign` regularly to stay up to date
- When ready, create a pull request from `uiux-redesign` to `main` for review and deployment

### How to use this branch
- Check out this branch locally: `git checkout uiux-redesign`
- Make and commit your UI/UX changes
- Push to `origin/uiux-redesign`
- Open a pull request to merge into `main` when the redesign is ready
